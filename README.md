# foodDeliveryWebsite
Food Delivery Website Project
Description
This is a simple food delivery website built using HTML, CSS, and JavaScript.
Users can view food items, add them to a cart, and see the total price update automatically.

Goals
Practice HTML, CSS, and JavaScript
Learn how to use JavaScript to update the page (DOM)
Build a simple cart system
Features
View food items
Add items to cart
Remove items from cart
Total price calculation
Responsive design

Technologies

CSS
JavaScript
